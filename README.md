# AI Resume Analyzer Web App

This is a full stack web application that analyzes resume text and provides skill detection, suggestions, and job role recommendations.

---

## 📌 Overview
A simple full stack project built using Flask that allows users to input resume text and receive analysis based on detected skills.

This project demonstrates basic backend logic, frontend interaction, and real-time analysis.

---

## 🚀 Features
- Web-based resume input interface  
- Skill detection (Python, ML, Data Analysis, Java)  
- Suggests relevant job roles  
- Provides improvement suggestions  
- Handles empty input validation  

---

## 🛠 Tech Stack
- Python  
- Flask  
- HTML & CSS
