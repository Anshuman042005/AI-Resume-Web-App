# AI Resume Analyzer Web App

This is a full stack web application that analyzes resume text and provides skill detection and job role suggestions.

---

## 🚀 Features
- Input resume text through web interface  
- Detects key skills like Python and Machine Learning  
- Suggests suitable job roles  
- Displays results dynamically on webpage  

---

## 🛠 Tech Stack
- Python  
- Flask  
- HTML
